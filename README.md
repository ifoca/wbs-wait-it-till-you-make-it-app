# Wait-It-Till-You-Make-It application

Full-stack application with React (Vite) frontend and Express backend. Helps you get a real time departure times for buses and trams of your favorite local station.

## Tech Stack

### Frontend (Client)

- React 19
- TypeScript
- Vite
- Tailwind CSS
- DaisyUI

### Backend (Server)

- Express
- MongoDB
- Mongoose
- Zod (validation)
- JWT (authentication)
- TypeScript

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- MongoDB (local or Atlas)
